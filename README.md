## üöÄ Cara Menjalankan
#### Requirements:
- Laptop
- Koneksi Internet
- Text Editor (Rekomendasi Visual Studio Code)
- Extension Five Server (Rekomendasi) / Live Server

#### Langkah-langkah
1. Unduh projek ke dalam storage masing-masing
```bash
git clone https://github.com/Falrafa4/thenews.git
```

2. Masuk ke folder "thenews"
```bash
cd thenews
```

3. Buka di Text Editor (Rekomendasi: Visual Studio Code)
```bash
code .
```

4. **Salin** file ```config.example.js``` yang terletak di dalam folder ```assets/js/```, lalu **rename** menjadi ```config.js```

5. Ubah API_KEY sesuai yang dimiliki.  
Note: Cara mendapatkan API_KEY adalah dengan mendaftarkan akun terlebih dahulu di https://newsapi.org/register

6. Jalankan Five Server / Live Server
7. Akses website di http://127.0.0.1:5000
